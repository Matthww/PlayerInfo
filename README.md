# PlayerInfo
Shows information about a certain player!

#### Command Usage:
`/playerinfo <player>`

We also have a shorter command:\
`/pinfo <player>`

#### Permission:
`playerinfo.use` - Allows the user/player to use the command `/playerinfo`

#### Example:

Name: Steve\
Model: iPhone\
OS: iOS\
IP: 127.0.0.1\
